# Changes

- Header based routing can now be used for [multiple rules](https://rollouts-plugin-trafficrouter-gatewayapi.readthedocs.io/en/latest/features/header-based-routing/#using-multiple-rules-in-a-single-route) within a single HTTP/GRPRoute
- Logging format can be [set to JSON](https://rollouts-plugin-trafficrouter-gatewayapi.readthedocs.io/en/latest/installation/#log-format)
- Cut down excessive logging for non-applicable resources (e.g. TLSRoutes)
